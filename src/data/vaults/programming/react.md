# React

React is a JavaScript library for building user interfaces through a component model and declarative rendering.

## Core Ideas

- **Components** — reusable, composable UI units (functions or classes)
- **JSX** — HTML-like syntax compiled to `React.createElement` calls
- **Unidirectional data flow** — props flow down, events flow up
- **Virtual DOM** — React diffs a virtual tree and applies minimal DOM updates
- **Hooks** — stateful logic in function components (introduced in v16.8)

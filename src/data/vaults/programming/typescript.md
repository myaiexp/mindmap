# TypeScript

TypeScript adds optional static typing to JavaScript, catching errors at compile time rather than runtime.

## Key Features

- **Type inference** — often infers types without annotations
- **Structural typing** — types are compatible if their shapes match
- **Generics** — write reusable, type-safe code
- **Decorators** — metadata annotations (experimental)
- **Strict mode** — `strict: true` enables all strictness checks
